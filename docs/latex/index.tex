Este proyecto consiste en un sistema de monitoreo que utiliza un sensor de temperatura y un sensor de ozono. Los datos son enviados desde un dispositivo Arduino a una aplicación frontend, donde se visualizan en tiempo real. La aplicación se comunica con un servidor que almacena y procesa la información.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Tabla de Contenidos}{Tabla de Contenidos}}\label{index_autotoc_md1}

\item Monitoreo en tiempo real de los niveles de ozono y temperatura.
